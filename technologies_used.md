# Technologies Used in Smart Courier Delivery Management System

Below is a comprehensive list of all the technologies, frameworks, and libraries used in the Blinkship microservices application, categorized by their roles.

## 1. Backend Core & Microservices
*   **Java 17**: Technology (Programming Language)
*   **Spring Boot (3.5.13)**: Framework (Core application framework for microservices)
*   **Spring Cloud Netflix Eureka (2025.0.1)**: Framework/Tool (Service Registry and Discovery)
*   **Spring Cloud Config**: Framework (Centralized Configuration Management)
*   **Spring Cloud Gateway**: Framework (API Gateway / Routing)
*   **Spring Cloud OpenFeign**: Library/Framework (Declarative REST Client for inter-service communication)
*   **Spring Data JPA**: Framework (ORM framework for database operations)
*   **Spring Security**: Framework (Authentication and authorization security)
*   **Spring Boot Actuator**: Library/Tool (System health and metrics monitoring)

## 2. Databases & Messaging
*   **PostgreSQL (15)**: Technology (Relational Database Management System)
*   **RabbitMQ**: Technology (Message Broker / Queuing system)

## 3. DevOps, Observability, & Security Tools
*   **Docker & Docker Compose**: Technology/Tool (Containerization and orchestration)
*   **Zipkin**: Tool (Distributed tracing system)
*   **Prometheus**: Tool (Metrics monitoring and alerting engine)
*   **SonarQube**: Tool (Static code analysis and code quality monitoring)
*   **Micrometer (Registry Prometheus & Tracing Bridge Brave)**: Library (Vendor-neutral application metrics facade)
*   **Zipkin Reporter Brave**: Library (Reporter for Zipkin tracing)
*   **JJWT (io.jsonwebtoken)**: Library (JSON Web Token generation and validation)
*   **BCrypt**: Library/Algorithm (Password hashing)
*   **Springdoc OpenAPI (Swagger UI)**: Library/Tool (API documentation generation)

## 4. Document & Image Processing (Backend)
*   **ZXing (Zebra Crossing)**: Library (QR code generation)
*   **iText HTML2PDF**: Library (Converts HTML templates to PDF format for invoices)

## 5. Frontend & UI
*   **Angular (21)**: Framework (Component-based frontend web framework)
*   **TypeScript (5.9)**: Technology (Strongly typed superset of JavaScript)
*   **Node.js (20+)**: Technology (JavaScript runtime environment for building frontend)
*   **Tailwind CSS (3.4.19)**: Framework (Utility-first CSS styling framework)
*   **RxJS**: Library (Reactive programming library for asynchronous data streams)

## 6. Build & Miscellaneous Tools
*   **Maven**: Tool (Backend build automation and dependency management)
*   **npm (Node Package Manager)**: Tool (Frontend dependency management)
*   **Lombok**: Library (Reduces Java boilerplate code like getters and setters)
*   **PowerShell / Bash**: Technology (Scripting languages for automation scripts)
